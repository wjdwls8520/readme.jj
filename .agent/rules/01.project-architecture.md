# General Coding Preferences (User: JeongJin)

1. **Strict Typing**: TypeScript `any` type is strictly forbidden.
2. **Naming**: 
   - Files: `kebab-case` (e.g., `user-profile.ts`)
   - Classes/Components: `PascalCase` (e.g., `UserProfile`)
   - Variables/Methods: `camelCase` (e.g., `getUserData`)
3. **Clean Code**:
   - Prefer `Early Return` pattern to avoid deep nesting.
   - Don't use `setter` methods in Java entities; use meaningful business methods (e.g., `updateStatus()`).
4. **Environment**: 
   - I use a **MacBook Air M2**. Ensure scripts are compatible with macOS (Unix-based).
